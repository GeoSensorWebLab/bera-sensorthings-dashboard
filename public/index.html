BERA SensorThings API Dashboard

SensorThings is an open specification for accessing geo-located sensor data. This dashboard allows you to browse any open SensorThings instance and visualize the results. It is built using the Eclipse Whiskers library.

This site requires JavaScript to be enabled in your browser.
